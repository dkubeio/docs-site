# Langflow on DKubeX — User Documentation

Welcome to the user documentation for Langflow on DKubeX. This covers everything you need to build, run, and deploy AI workflows on the platform.

## Contents

- [Getting Started](./getting-started.md) — Log in, launch Langflow, and create your first flow.
- [Building Flows](./building-flows.md) — Author flows visually using the canvas, configure components, and manage secrets.
- [Components](./components.md) — Overview of the built-in component catalog, including the DKubeX Providers (SecureLLM-backed LLM and embedding components).
- [Deploying Flows](./deploying-flows.md) — Promote a flow to a standalone, always-on API endpoint managed by DKubeX.
- [API Usage](./api-usage.md) — Call your flow programmatically, either through the main Langflow instance or a dedicated deployed pod.
- [FAQ](./faq.md) — Frequently asked questions and troubleshooting tips.
