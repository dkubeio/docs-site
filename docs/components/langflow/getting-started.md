# Getting Started

Langflow is a visual workflow builder for AI-powered agents and pipelines. This guide walks you through launching Langflow and running your first flow.

## Prerequisites

- Access to a DKubeX environment with Langflow enabled, or a local Langflow install.
- A modern browser (Chrome, Firefox, Safari, or Edge).

## Launching Langflow

1. Open your DKubeX dashboard and click the **Langflow** tile, or navigate directly to `/langflow/`.
2. The Langflow canvas loads with the DKubeX header pinned to the top.
3. Use the **Development Mode** / **Deployment Mode** toggle in the header to switch contexts.

## Creating Your First Flow

1. Click **New Flow** from the projects list.
2. Drag components from the left sidebar onto the canvas.
3. Connect component outputs to inputs by dragging between the colored ports.
4. Click the **Play** button on any component to run the flow up to that point.
